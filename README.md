# XoomScript
Язик програмириванние исползуйщий компилятор на пайтоне похожий на асемблер оринтириваний на опенсоурс

что надо
1 пайтон установлений
2 Библиотека LLVM: pip install llvmlite
3 Для сборки EXE: Установить LLVM/Clang (чтобы команда clang была доступна в системе).

Основные команды (Терминал)
для манипуляций надо находитса в папке с файлам main.py заходите в папуку где маин пай и правая кнопка миши и зажатий шифт и открвиаете через терминалл павершелл
1 python main.py --version показиваеть версиюу
2 python main.py run main.xs запускаеть фаил с кодом ВНИМАННИЕ КОД ДОЛЖЕН ИМЕЕТЬ РАШИРЕННИЕ XS ИНАЧЕ НЕЧЕГО НЕ ЗАПУСТИТСА
3 python main.py build main.xs name=myapp билдитьфаил в ехе но для этого надо имееть clang установлений на пк 

синтексис и функции
1 работа с даними
move int 100 >> rax;      ### Положить 100 в rax
add int 50 >> rax;       ### rax = 150
clear [rax, rbx];         ### Обнулить регистры
2 логика и условие
check (rax == 0) = printconsole("Zero!");   ### Выполнить, если rax равен 0
check (rbx >= 10) = jump mentos=label;      ### Прыгнуть на метку, если rbx >= 10
3 цикли ии переход
set mento=start;          ### Создать точку входа
... код ...
jump mentos=start;        ### Вернуться в начало (бесконечный цикл)
4 водд и вивод
input("Vvedite ID: ");    ### Напечатать текст и ждать ввода
input_save >> r8;         ### Сохранить введенное число в r8
printconsole("Result: ", r8, 5+5); ### Вывод текста, регистра и математики
